# ⚡ FlashAI v1.0

텍스트를 붙여넣으면 AI(Claude)가 Q&A 플래시카드를 생성해주는 웹 앱입니다.
**프로세스를 적용하지 않고 빠르게 바이브코딩한 초기 버전**입니다.

---

## 📌 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 과제 | 프로세스에 입각한 바이브코딩의 효과 분석 |
| SW | FlashAI — AI 기반 Q&A 플래시카드 생성기 |
| 버전 | v1.0 (프로세스 미적용 / 빠른 프로토타입) |
| 사용 AI | Claude (Anthropic, `claude-sonnet-4`) |
| 개발 도구 | HTML / CSS / JavaScript (단일 파일) |
| 팀원 | 나샤론 신주영 |

---

## ⚠️ 이 버전의 성격

v1.0은 **요구사항 분석·설계·테스트 등의 프로세스 없이**, AI에게 바로
"카드 만들어줘"라고 요청해 빠르게 만든 버전입니다. v2.0(프로세스 적용)과
비교하기 위한 **대조군** 역할을 합니다.

| 특징 | 내용 |
|------|------|
| 토큰 표시 | 한 줄로 단순 표시 |
| 입력 검증 | 거의 없음 (키·텍스트 존재 여부만) |
| 에러 처리 | 없음 (실패 시 그냥 멈춤) |
| 옵션 | 없음 (카드 5개 고정) |
| 초기화 | 페이지 새로고침 |
| 코드 규모 | 약 70줄 |

---

## ✅ 기능 (v1.0)

- AI 카드 자동 생성 (5개 고정)
- 카드 클릭 시 답 표시
- 사용 토큰 수 한 줄 표시

---

## 🚀 실행 방법

1. `index.html`을 브라우저로 열기
2. Anthropic API 키(`sk-ant-...`) 입력
3. 학습할 텍스트 붙여넣기
4. **카드 생성** 클릭

> 🔑 API 키는 [console.anthropic.com](https://console.anthropic.com)에서 발급받습니다.

---

## 🛠️ 사용 기술

- HTML / CSS / JavaScript (Vanilla)
- Anthropic Claude API (`claude-sonnet-4-20250514`)

---

## 🔄 v2.0과의 비교

프로세스를 적용한 개선 버전은 별도 폴더(`flashai-v2`)를 참고하세요.

| 항목 | v1.0 (현재) | v2.0 (프로세스 적용) |
|------|------------|---------------------|
| 토큰 표시 | 단순 | 대시보드 + 비용 계산 |
| 생성 이력 | ❌ | ✅ 로그 테이블 |
| 입력 검증 | ❌ | ✅ 길이 체크 |
| 에러 처리 | ❌ | ✅ 자동 재시도 |
| 옵션 | ❌ | ✅ 개수·난이도 |
