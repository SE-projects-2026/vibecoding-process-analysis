# ⚡ FlashAI v2.0

텍스트를 붙여넣으면 AI(Claude)가 자동으로 Q&A 플래시카드를 생성해주는 웹 앱입니다.
**프로세스에 입각한 바이브코딩**을 적용하여 개발한 버전입니다.

---

## 📌 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 과제 | 프로세스에 입각한 바이브코딩의 효과 분석 |
| SW | FlashAI — AI 기반 Q&A 플래시카드 생성기 |
| 사용 AI | Claude (Anthropic, `claude-sonnet-4`) |
| 개발 도구 | HTML / CSS / JavaScript (단일 파일) |
| 팀원 | 나샤론 신주영 |

---

## 🔄 버전 비교 (핵심)

본 프로젝트는 **프로세스 미적용(v1.0)** 과 **프로세스 적용(v2.0)** 을 비교합니다.

| 항목 | v1.0 (그냥 바이브코딩) | v2.0 (프로세스 적용) |
|------|----------------------|---------------------|
| 토큰 표시 | 한 줄 단순 표시 | 대시보드 + 비용 자동 계산 |
| 생성 이력 | ❌ 없음 | ✅ 로그 테이블 기록 |
| 입력 검증 | ❌ 없음 | ✅ 길이 체크 (20~5000자) |
| 에러 처리 | ❌ 실패 시 멈춤 | ✅ 최대 2회 자동 재시도 |
| 사용자 옵션 | 카드 5개 고정 | 개수(3~10) · 난이도 선택 |
| 코드 규모 | 약 70줄 | 약 400줄 |

---

## ✅ 주요 기능 (v2.0)

- **AI 카드 자동 생성**: 입력 텍스트 기반 Q&A 플래시카드 생성
- **카드 뒤집기**: 클릭 시 앞면(질문)/뒷면(답변) 전환
- **토큰 대시보드**: 입력 / 출력 / 누적 토큰 + 예상 비용(USD) 실시간 표시
- **생성 이력 로그**: 시각·글자수·카드수·토큰·성공여부 기록 (프로세스 증빙)
- **입력 검증**: 최소 20자, 최대 5000자 제한 및 실시간 글자수 표시
- **자동 재시도**: JSON 파싱 실패 시 최대 2회 재시도
- **옵션 설정**: 카드 개수(3/5/7/10), 난이도(기초/보통/심화)

## 🚀 실행 방법

별도 설치 없이 `index.html` 파일을 브라우저에서 열면 실행됩니다.

1. `index.html`을 브라우저로 열기
2. Anthropic API 키(`sk-ant-...`) 입력 → **시작**
3. 학습할 텍스트 붙여넣기
4. 카드 개수·난이도 선택 후 **카드 생성하기**

> 🔑 API 키는 [console.anthropic.com](https://console.anthropic.com)에서 발급받을 수 있습니다.
> 키는 브라우저 메모리에만 저장되며 코드/서버로 전송되지 않습니다.
