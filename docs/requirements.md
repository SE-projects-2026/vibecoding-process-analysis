# 요구사항 — FlashAI v2.0

## v1.0 문제점
- API 키 하드코딩 → 보안 위험
- 입력 길이 검증 없음 → 빈 입력도 API 호출
- 카드 수 5개 고정 → 사용자 선택 불가
- 오류 시 그냥 멈춤 → 재시도 없음
- 토큰 합계만 표시 → 비용 파악 불가

## v2.0 개선 사항
- API 키 실행 시 입력 (하드코딩 제거)
- 입력 20자~5000자 범위 검증
- 카드 수 3/5/7/10개 선택 가능
- 오류 시 최대 2회 자동 재시도
- 토큰 입력/출력/누적/비용 대시보드
- 생성 이력 로그 테이블
- 난이도 기초/보통/심화 선택
