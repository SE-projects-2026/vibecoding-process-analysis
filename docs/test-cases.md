## 1. 기능 테스트

### TC-01: 정상 카드 생성 (F-01)
| 항목 | 내용 |
|------|------|
| **목적** | 카드 수 선택 후 정확한 수의 카드 생성 확인 |

## 4. v1.0 vs v2.0 품질 비교

| 항목 | v1.0 | v2.0 | 개선 |
|------|------|------|------|
| 입력 검증 | ❌ 없음 | ✅ 30자 이하 경고 | 불필요한 API 호출 방지 |
| 카드 수 선택 | ❌ AI 임의 결정 | ✅ 사용자 선택 (3/5/7) | 사용자 통제권 확보 |
| 학습 상태 추적 | ❌ 없음 | ✅ ✅/❌ + 진도 바 | 반복 학습 지원 |
| 오류 메시지 | ❌ 단순 메시지 | ✅ 원인별 안내 | 사용자 혼란 감소 |
| 로딩 피드백 | ⚠️ 텍스트만 | ✅ 애니메이션 | 사용자 경험 개선 |
| XSS 방지 | ❌ 없음 | ✅ escHtml() 적용 | 보안 강화 |
| 모바일 대응 | ⚠️ 불안정 | ✅ 미디어쿼리 추가 | 접근성 개선 |
